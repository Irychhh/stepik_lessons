Oтличительные особенности у кода, экспортированного из рекордера, по сравнению с кодом, написанным вручную

1. Такой код может быть использован для записи простых, прямолинейных автоматизированных тестов без умения программировать.
2. Не умеет работать со "сложной логикой", "если сделать так, то будет вот так, иначе..".
3. Нет assertов.
4. Хорош для помощи и адаптации новичку в автоматизации, для разбора и анализа кода.
5. Не нужно искать локаторы на странице (допустим, на странице, где если их бывает сложно подобрать).
6. Искусственный код понятен, линеен и легко читается, есть комментарии.
7. Чтобы получить код, не требуется дополнительной настройки среды (кроме установки Selenium IDE).
8. Вряд ли такие тесты стабильны.
9. Такой нельзя интегрировать в общий проект.
